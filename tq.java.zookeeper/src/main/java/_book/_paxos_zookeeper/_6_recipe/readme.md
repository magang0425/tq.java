2. 全局配置信息
    - 数据量小
    - 动态变化
    - 集群同步
3. 总计为
    - 配置存储
    - 配置获取

1. 命名服务的资源定位 仅仅 是一个资源标识

2. 分布式全局唯一ID的分配机制
    - UUID
        - 太长
        - 含义不明
3. 顺序节点
    - 并且在 api返回值中可以获取这个节点的完整名字
